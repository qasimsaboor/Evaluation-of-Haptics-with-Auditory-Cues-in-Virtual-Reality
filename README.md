#Build Versions 
#How to run Build Versions?
- Download the Haptics Build Version or Non Haptics Build Version folder from git. 
- Download Steam, install and login to Steam.
- Connect HTC Vive Pro Eye, Base Stations and Sense Gloves DK1 with HTC Vive Trackers 3.0 to the PC.
- Download and install and run SenseCom which is downloaded using this API link (Link : https://github.com/Adjuvo/) and run it. When the
  gloves show up as blue icons within the SenseCom UI, Sense Gloves are connected. 
- Download Steam VR and run it. Create SteamVR Input Bindings and select Vive Trackers in Hands as Controller.
- All the connected devices will be visible in SteamVR. 
- Run "Prototype 2.1.exe" file from the build folder.
- Wear the HTC Vive Pro Eye with Sense Gloves DK1 and enjoy!


#Source Code
#How to run Source Code?
- Download Unity3d v2020.3.19f1 and login to Unity Hub.
- Download the Haptics or Non Haptics folder from git and open it as seperate projects in Unity3d.
- Download Steam and Login to Steam.
- Login to Unity, download SteamVR from Unity Asset Store and import it. 
- Connect HTC Vive Pro Eye and Sense Gloves DK1 with HTC vive trackers 3.0. Set the base stations and callibrate them. 
- Download Sense Gloves SDK v2.5.0 and import it in the Unity project using this link. 
  (Link : https://github.com/Adjuvo/SenseGlove-Unity/blob/master/Older%20Versions/SenseGlove_Unity_v2_5_0.unitypackage).
- Download and install and run SenseCom which is downloaded using this API link (Link : https://github.com/Adjuvo/) and run it. When the
  gloves show up as blue icons within the SenseCom UI, Sense Gloves are connected. 
- Create SteamVR Input Bindings and select Vive Trackers in Hands as Controller.
- All the connected devices will be visible in SteamVR. 
- Run the Project in Unity3d and wear the HTC Vive Pro Eye and Sense Gloves with HTC vive trackers on them to test it. 

#Data Analysis Scripts 
#How to run Data Analysis Scripts?
- Download python v3.10.9.
- Download Anaconda v2023.03-1 and install it.
- From Anaconda open jupyter notebook and open the "Masters Project Data Analysis" file in it
- Download haptics and non haptics csv data files from git to specific folder.
- In "Masters Project Data Analysis" file give path the csv data file where these are stored and run the code, it will display the output.
